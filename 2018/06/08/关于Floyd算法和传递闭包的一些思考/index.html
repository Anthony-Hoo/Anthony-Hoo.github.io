    关于传递闭包

    在数学上，传递闭包的定义是：

    传递闭包、即在数学中，在集合 X 上的二元关系 R 的传递闭包是包含 R 的 X 上的最小的传递关系。
    例如，如果 X 是(生或死)人的集合而 R 是关系“为父子”，则 R 的传递闭包是关系“x 是 y 的祖先”。再比如，如果 X 是空港的集合而关系 xRy 为“从空港 x 到空港 y 有直航”，则 R 的传递闭包是“可能经一次或多次航行从 x 飞到 y”。 –wikipedia

    Floyd算法和传递闭包的关系

    从以上的伪代码和解决问题的过程看来，Floyd算法与Warshall算法十分相似（只有一行代码的差别）。而实际上，求解一个图的任意两节点间的最短路，就需要将这两个节点间的所有路径全部罗列出来，再进行比较。这一点上，Warshall算法实际上已经将任意有通路的节点连接了起来，Floyd算法要做的就是将最短的那一条路径找出来罢了。因此，Floyd算法常被看作Warshall算法的一个扩展，故其也名为Floyd-Warshall算法。
