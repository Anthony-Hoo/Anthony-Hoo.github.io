    Section 1: Propositional Logic

    This section is mainly about propositions and logical operators .

    Propositions

    A proposition is a declarative sentence that is either true or false, but not both.
    For an example :

    1) What time is it?
    2) Read this carefully.
    3）x + y = z
    4) 1 + 1 = 2
    5) Toronto is the capital of Canada

    1) and 2)are not propositions because they are not declarative sentence, 3) is not propositions because we can’t tell it true or false since the value of the variables x y and z are unknown .
    Both 4) and 5) are propositions even though 5) is not true. So we call 4) is a true proposition and 5) is a false proposition .

    Tips

    The one confuse me most in the five logical operators is the conditional statement. Especially in the case p = FALSE and q = TRUE, the statement p ⟶ q is not false but true，think about it： a false premise can infer into a true conclusion, it’s puzzling, isn’t it?
    In the later study, I happened noticed this in www.zhihu.com

    可以这样理解:假如小明的爸爸对小明承诺，只要小明离散数学考了100分，就给他买 iPhone7。这个例子中， p是“小明离散数学考了100分”， q是“小明爸爸给小明买 iPhone7”。假如 p为真，也就是小明考了100分，如果此时 q为假，那爸爸的承诺 p-> q就为假。假如 p为假，那无论小明他爸给不给他买 iPhone7，都不会违背那个承诺，也就是 p-> q为真。因此，只有当 p为真， q为假时， p-> q才为假。

    English ver：

    It can be understood as follows: If Xiao Ming’s father promised Xiao Ming, as long as Xiao Ming’s discrete math test scored 100 points, he would buy him iPhone7.
    In this example, p is “Xiaoming discrete mathematics test 100 points”, q is “Xiao Ming dad to buy Xiaoming iPhone7”.
    If p is true, that is, Xiao Ming took 100 points. If q is false at this time, then Dad’s promise p-> q is false. If p is false, then no matter if Xiao Ming does not give him an iPhone7, he will not violate the promise, that is, p->q is true. Therefore, p->q is false only if p is true and q is false.

    Therefore, in the case of p being false, the truth of the proposition is not related to whether q is true or not.
